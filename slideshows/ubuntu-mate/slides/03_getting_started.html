Get up and running faster

Ubuntu MATE Welcome greets and assists new and returning users. The Getting Started wizard will guide you through the first boot post-installation setup and the configuration of additional peripherals and devices, while the documentation will help orientate new users. The Systems Specifications section will help you explore your hardware and offers one-click installation for many useful system monitoring tools. When you need help Ubuntu MATE Welcome is the jumping off point to the community.
