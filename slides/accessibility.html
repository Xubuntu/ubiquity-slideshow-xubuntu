Accessibility in Ubuntu

  • One of our goals is to make computing easy whatever peoples' circumstances. So, we provide a range of tools that make Ubuntu one of the most accessible operating systems around.
  • You can get at these tools in one place: the Assistive Technologies Preferences. From there, you can set up wrist-friendly behaviours for the mouse and keyboard. The Preferred Applications section helps to turn on a screen reader, like Orca, or an on-screen keyboard.
  • There are some handy options in Appearance Preferences, too. For example, to make text more readable, you can customize the fonts that are used by applications.
